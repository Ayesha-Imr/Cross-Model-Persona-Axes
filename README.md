# Cross-Model Persona Axes

WIP: Probing how different LLMs exhibit various persona traits (sycophancy, warmth, bluntness, refusal-stiffness, and more) in normal interactions by projecting their responses through a shared Gemma 4 activation space onto interpretable behavioral axes. Inspired by [this experiment](https://x.com/_lyraaaa_/status/2046078076027884014?s=20).
